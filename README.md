## Dependencies
- Python 3.x
- Tkinter
- Numpy
## How To Run
1. Download the latest released version of the game
2. Open the game directory containing `main.py`
3. Run `main.py` (make sure you are in the game directory)
## Controls
- Arrow keys & WASD = Movement
- Undo Button = Undo the last move
- Save Button = Save current session as .txt file
- Load Button = Load a session
