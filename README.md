## Dependencies
- Tkinter
- Numpy
## How To Run
- Download the last released version
- Open the game directory containing main.py
- Run main.py as you are in game directory
## Controls
- Arrow keys & WASD = Movement
- Undo Button = Undo a move
- Save Button = Save current session as .txt file
- Load Button = Load a session
